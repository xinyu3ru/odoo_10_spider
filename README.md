# odoo_10_spider
 python 读取odoo 10 客户信息，修改信息，导出联系方式


 
 ## 请勿非法使用，不承担任何非法使用带来的责任
 
 

 
 某公司使用odoo 10 社区版管理生产和客户资料，作为销售人员，需要把客户联系方式导入手机，为减轻重复劳动，写此脚本。
 
 ***请勿非法使用，不承担任何非法使用带来的责任。***
 
 在odoo网页界面可以通过复制json数据或者Elements数据，处理数据之后，可以得到需要的联系方式，相对比较麻烦。不会编程可以尝试这样的方式。
 
 某公司又经常调换客户，需要改写odoo系统中客户归属，写脚本实现快速改写，是一个不错的选择。
